Installation
===========================================

1. Create a conda environment and activate it.

.. code-block:: python
   
   conda create --name pysodb python=3.8 -y
   conda activate pysodb


2. Install pysodb as a dependency or third-party package with pip:

.. code-block:: python
   
   cd pysodb_package
   python setup.py install
